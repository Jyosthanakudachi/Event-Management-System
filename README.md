# 🎉 Event Management System

## 📌 Overview
Event Management System is a web-based application developed to simplify event planning, booking, and management. The system allows users to browse events, register, book events, and manage event details through an intuitive interface.

## 🚀 Features
- User Registration and Login
- Event Browsing and Booking
- Event Categories (Wedding, Engagement, Reception, Birthday, Festivals, etc.)
- Admin Dashboard
- Booking Management
- User Authentication
- Responsive User Interface

## 🛠️ Technologies Used

- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
